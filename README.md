# mutfundtool
To run this program, you need to install some of the libraries. You will need to install yfinance, pandas, numpy, and plotly in the terminal of the IDE. I used Visual Studio code so I don't know if it will be different for other IDEs

All you need to do is pip install yfinance pandas numpy plotly and then the code should work. My code does graph the charts in a web browser and should open your default browser to do so. Otherwise, thanks for an amazing semester. Have a great summer!
